IAAS

IaaS oferece infraestrutura de TI virtualizada sob demanda, incluindo servidores, armazenamento, redes e sistemas operacionais. Os usuários têm controle total sobre os recursos, podendo instalar e gerenciar seus próprios sistemas operacionais e aplicações.
Exemplo: Amazon Web Services (AWS) EC2, Google Compute Engine, Microsoft Azure.

PAAS

PaaS fornece uma plataforma que permite aos desenvolvedores criar, testar e implantar aplicações rapidamente sem se preocupar com a infraestrutura subjacente. Inclui ferramentas de desenvolvimento, bancos de dados, servidores de aplicação, e ambientes de execução.
Exemplo: Google App Engine, Microsoft Azure App Service, Heroku.

SAAS

SaaS oferece software acessível via internet, geralmente através de um navegador. Os usuários não precisam instalar ou manter o software, pois ele é gerenciado pelo provedor de serviços. É o modelo mais comum para aplicativos de negócios e consumidores.
Exemplo: Gmail, Microsoft Office 365, Salesforce.
